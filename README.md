# Employee Attendance Dashboard (Power BI)

## Project Overview
This project is an interactive Power BI dashboard designed to analyze employee attendance and team performance.

## Features
- Total employees enrolled
- Present and absent employee analysis
- Average attendance rate
- Team-wise attendance insights
- Manager-wise performance analysis

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

![Dashboard](dashboard.png)

## Insights
The dashboard helps managers quickly understand attendance trends and identify team performance patterns.
